# Tagdown JS --- ### Usage * Include the script sources in the bottom of your HTML file. * `<script src="https://spikespaz.github.io/tagdownjs/scripts/tagdown.min.js"></script>` * Place all Markdown text between `<markdown>` and `</markdown>` tags. ##### Copy the HTML template below and paste it into a file server: ```html <!DOCTYPE html> <html> <title>Hello Tagdown</title> <markdown> # Markdown text goes in here ## Chapter 1 Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. ## Chapter 2 Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum. </markdown> <script src="https://spikespaz.github.io/tagdownjs/scripts/tagdown.min.js"></script> </html> ``` * ***Modify untill your hearts content.*** To see a working example of the above code, click [here](example.html). --- ### File Information: > **Filenames:** "`tagdown.js`" and "`tagdown.min.js`" > **Author:** "spikespaz" > **Email:** "spikespaz@outlook.com" > **GitHub:** "github.com/spikespaz/tagdownjs" > **Website:** "https://spikespaz.github.io/tagdownjs" > **Description:** "A script to dynamically render Markdown text as HTML." --- ### Includes Markdown Parser: > "`marked.js`" by *Christopher Jeffrey* > `Copyright (c) 2011-2014, Christopher Jeffrey.` *(MIT Licensed)* > **GitHub URL:** https://github.com/chjj/marked --- ### Script Sources: * #### [Source Script](scripts/tagdown.js) * #### [Minified Script](scripts/tagdown.min.js) --- `TagdownJS - Copyright Â© 2016, spikespaz (MIT Licensed)` - `Paper theme from Bootswatch.com - by Thomas Park`
