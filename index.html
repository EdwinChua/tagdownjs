### About Tagdown JS is a simple javascript file that can be linked as a script source to render Markdown to HTML dynamically, without the need for any server-side conversion. Dynamic rendering allows for faster writing and documentation via GitHub Pages. **Please note** that any section of Markdown text cannot contain the string `</markdown>` unless it is enclosed in `<xmp>` tags. All of the text that you see on the main body of this page was written in Markdown and rendered into HTML by the script. Two scripts are provided: * `tagdown.js` *(standard)* * Readable, clean code * `tagdown.min.js` *(minified)* * Faster rendering * Faster page loading * Smaller file size If you are just going to be using the script without modificaion, link `tagdown.min.js` *(minified)* for rendering. Original script minified using [UglifyJS2](https://github.com/mishoo/UglifyJS2). --- ### Usage * Include the script source in the bottom of your HTML file. * `<script src="https://spikespaz.github.io/tagdownjs/scripts/tagdown.min.js"></script>` * Place all Markdown text between `<markdown>` and `</markdown>` tags. ##### Copy the HTML template below and paste it into a file server: ```html <!DOCTYPE html> <html> <title>Hello Tagdown</title> <markdown> # Markdown text goes in here ## Chapter 1 Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. ## Chapter 2 Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum. </markdown> <script src="https://spikespaz.github.io/tagdownjs/scripts/tagdown.min.js"></script> </html> ``` To see a working example of the above code, click [here](example.html). --- ### File Information > **Filenames:** "`tagdown.js`" and "`tagdown.min.js`" > **Author:** "spikespaz" > **Email:** "spikespaz@outlook.com" > **GitHub:** "github.com/spikespaz/tagdownjs" > **Website:** "https://spikespaz.github.io/tagdownjs" > **Description:** "A script to dynamically render Markdown text as HTML." --- > "`marked.js`" by *Christopher Jeffrey* > `Copyright (c) 2011-2014, Christopher Jeffrey.` *(MIT Licensed)* > **GitHub URL:** https://github.com/chjj/marked ---
